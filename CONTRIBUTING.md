# Contributing to Procedural Handwriting Generator

First off, thank you for considering contributing to this project! It's people like you that make open-source software such a great community. 

## 🐛 Reporting Bugs
If you find a bug in the code or a mistake in the documentation, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.

**Before submitting a bug report, please check:**
* Is the issue already reported? Search the existing issues.
* Are you using the latest version of the code?

## ✨ Suggesting Enhancements
We are always looking for ways to make the generated handwriting look more natural! If you have an idea for a new feature (like a new pen color, paper style, or a new procedural distortion method), please open an issue to discuss it before you begin writing code.

## 🛠️ Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Write clear, concise code** and adhere to standard Java coding conventions.
3. **Test your code** by generating a few sample outputs to ensure your changes don't break existing features.
4. **Update the README** if your changes add new menu options or require new dependencies.
5. **Issue that pull request!**

## 💻 Development Setup
Since this is a standard Java project with no external dependencies, setup is simple:
1. Clone the repo.
2. Ensure you have JDK 8+ installed.
3. Compile using `javac TextToHandwritten.java`.
4. Run using `java TextToHandwritten`.

Thank you for your contributions!
